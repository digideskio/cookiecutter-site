
=============
Leonardo Site
=============

Place especially for project migrations and other project specific stuff.


Installation
============

.. code-block:: bash

    source leonardo/sites/project_name/bin/activate

    manage.py makemigrations --noinput

    git add .
    git commit -m "Initial Commit"
    git push
